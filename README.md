### 📌 Sobre o projeto

Este projeto é um laboratório prático de autenticação fullstack, com foco em:

- Login seguro com JWT
- Gerenciamento de usuários
- Upload de avatar
- Proteção de rotas
- Integração completa entre frontend e backend

<hr>

### 🚀 Tecnologias

Backend:
- Node.js
- Express
- MongoDB
- JWT
- Bcrypt
- Multer

Frontend:
- React
- React Router DOM
- Axios
- Context API

<hr>

### 📦 Funcionalidades

Autenticação:
- Cadastro de usuário
- Login com email e senha
- Logout

Perfil do usuário:
- Visualizar dados do usuário logado
- Editar nome
- Editar email
- Upload de avatar

Segurança:
- Senha criptografada com bcrypt
- Autenticação com JWT
- Middleware de proteção de rotas
- Cookie httpOnly

Upload de imagem:
- Upload com multer
- Salvar caminho da imagem no banco
- Exibir avatar no frontend

Frontend:
- Tela de login
- Tela de cadastro
- Dashboard (área logada)
- Página de perfil
- Proteção de rotas

<hr>

### 🎯 Objetivo

Este projeto foi criado para praticar:

- ✅ Autenticação com JWT
- ✅ Estrutura profissional de backend

- ✅ Segurança em aplicações web
